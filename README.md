# Scoreboard

LunaChat と Discord をチャット連携する BungeeCord プラグインです。

## Requirements

- Java 11+
- BungeeCord
- [LunaChat 3.0.8+](https://github.com/ucchyocean/LunaChat/releases/tag/v3.0.8)

## Usage

サーバーディレクトリの `/plugins/` に配置し、サーバーを再起動する。

作成された `config.yml` にて各種設定する。
